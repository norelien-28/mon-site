Introduction au DOM (Document Object Model)

Le DOM est une interface de programmation pour les documents HTML et XML.

Il représente la page web comme une structure d’objets que JavaScript peut manipuler.

Quelques actions courantes :

  • Sélectionner un élément : document.getElementById("monId")
  • Modifier le contenu : element.innerHTML = "Nouveau contenu";
  • Ajouter un écouteur d'événement : element.addEventListener("click", fonction);

Exemple :


const bouton = document.getElementById("monBouton");
bouton.addEventListener("click", () => {
  alert("Bouton cliqué !");
});

Pour en savoir plus, visite la documentation MDN DOM.
